# **App Name**: MotionFolio

## Core Features:

- Navbar with Smooth Scroll: A sticky navigation bar with animated mobile menu and smooth scrolling to different sections of the portfolio.
- Hero Section: A landing section with a creative heading, typewriter effect, animated profile, social icons with hover effects, CTAs, and a glowing background.
- About Section with Animated Profile Card: A section with a brief introduction, an animated profile card, and animated skills tags.
- Skills Section with Grid Layout: A grid layout showcasing various skills using skill cards with icons and staggered fade-in animations.
- Projects Section with Animated Cards: A projects section with modern project cards, hover effects, card animations, GitHub and Live Demo buttons, and a creative grid layout.
- Contact Form with Animated Fields: An animated contact form with fields that animate on focus, utilizing a Next.js API route for email sending.

## Style Guidelines:

- Primary color: Electric Purple (#BE29EC) to give a futuristic and modern look.
- Background color: Very dark grey (#121212), near black, providing a sleek backdrop.
- Accent color: Neon Green (#39FF14) used for highlights and interactive elements.
- Body and headline font: 'Space Grotesk', a proportional sans-serif, for a computerized, techy feel. If longer text is used for project descriptions or blog content, consider pairing with 'Inter' for body text.
- Use modern, minimalist icons to represent skills and technologies, with subtle hover animations.
- Implement a clean and modern layout with rounded corners, soft shadows, and glassmorphism effects.
- Apply smooth Framer Motion animations throughout the website, including stagger animations, hover effects, and transition animations.
